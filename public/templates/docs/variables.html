Global variables (available in any task)

  • project_name repository name
  • inventory_name inventory name
  • global Repository name
  • build_path path to build on dogpro
  • build_version commit hash
  • build_version_short commit short hash

Deploy variables (available in dogpro.deploy playbook)

  • release_name release id (20150101153030)
  • release_dir path to release in remote server
  • shared_dir path to shared directory in remote server
  • current_dir current release dir (/var/www/current)
  • previous_release path to previous release
  • assets_dir path to assets directory
  • prev_assets_dir path to previous release assets directory
